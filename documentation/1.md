
# NeuralSonic

  

## Idea
  To make a 2D game like Sonic Mania in Godot Game Engine. To train AI, which will play the game and earn points/rings.
  
  The game will have options like:
  - playing as a user, as a person
  - AI mode, where an AI plays and you see it
  - AI vs. person (for future development)

At first the game will have only three levels and one character. The game is gonna be loading with a Home screen in Sonic style, then a "Play" button, "AI mode" button, "Options" button and "Exit".
